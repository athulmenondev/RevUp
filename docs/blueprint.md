# **App Name**: IEEE Event Hub

## Core Features:

- Event Listing: Display a list of IEEE events with details such as date, time, location, description, speakers, and agenda.
- Event Detail Page: Dedicated page for each event with comprehensive information.
- Registration Form: Allow users to register for events through a form.
- Interactive Map: Display the location of the event venue through a tool that interfaces with a maps provider and marks the venue location

## Style Guidelines:

- Primary color: Deep blue (#293462), reflecting the professional and technical nature of IEEE.
- Background color: Light blue (#D6E4FF), providing a clean and modern backdrop.
- Accent color: Yellow (#FFD700) to highlight important information and CTAs.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable look.
- Use IEEE-themed icons related to technology, networking, and learning.
- Clean and responsive layout optimized for desktop and mobile devices.
- Subtle transitions and animations to enhance user experience.
